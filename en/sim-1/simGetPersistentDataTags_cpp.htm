simGetPersistentDataTags (in sim-1)

Deprecated. See simReadCustomDataTags instead

C++ synopsis

char* simGetPersistentDataTags(int* tagCount)

Arguments

  • tagCount: the number of retrieved data tags

Return

  • nullptr if operation was not successful or no data exists. Otherwise the data tags, separated by the '\0' char. In that case the returned buffer should be released with simReleaseBuffer
