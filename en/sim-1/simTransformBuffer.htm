sim.transformBuffer sim-1

Modifies a buffer than contains packed data

Synopsis

a) bytes outBuff = sim.transformBuffer(bytes inBuff, int inFormat, float mult, float off, int outFormat) b) list outBuff = sim.transformBuffer(bytes inBuff, sim.buffer_uint8, 0, int splitSize, sim.buffer_split) a) buffer outBuff = sim.transformBuffer(buffer inBuffer, int inFormat, float mult, float off, int outFormat) b) buffer[] outBuff = sim.transformBuffer(buffer inBuffer, sim.buffer_uint8, 0, int splitSize, sim.buffer_split)

Arguments

  • inBuff: input buffer that contains packed data
  • inFormat: buffer type
  • mult: multiplier value. We have out = offset + multiplier * in
  • off/splitSize: offset or split size value. We have out = offset + multiplier * in
  • outFormat: desired buffer type for the returned buffer

Return values

  • a) outBuff: the modified buffer
  • b) outBuff: the various split buffers
