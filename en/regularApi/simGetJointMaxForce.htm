simGetJointMaxForce / sim.getJointMaxForce

Description Retrieves the maximum force or torque that a joint can exert. See also sim.setJointMaxForce and sim.getJointTargetVelocity.
C synopsis simInt simGetJointMaxForce(simInt jointHandle,simFloat* forceOrTorque)
C parameters
jointHandle: handle of the joint.
forceOrTorque: the maximum force or torque the joint can apply.
C return value
-1 if operation was not successful. .
