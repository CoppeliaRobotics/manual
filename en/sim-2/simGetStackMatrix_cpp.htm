simGetStackMatrix (in sim-2)

Tries to retrieve the value at the top of the stack, if that value is a matrix

C++ synopsis

double* simGetStackMatrix(int stackHandle, int* rows, int* cols)

Arguments

  • stackHandle: a stack handle obtained with simCreateStack.
  • rows: a pointer to a location receiving the number of rows. Can be nullptr
  • cols: a pointer to a location receiving the number of columns. Can be nullptr

Return

  • in case of an error, the return value is nullptr and rows/cols are set to -1. If the stack item is not a matrix, the return value is nullptr and rows/cols are set to 0. The user is in charge of releasing the returned buffer with simReleaseBuffer)
