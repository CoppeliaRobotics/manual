C++ synopsis

int simWriteCustomDataBlock(int objectHandle, const char* tagName, const char* data, int dataSize)

Arguments

  • objectHandle: handle of the object where you want to store your data, sim.handle_scene, sim.handle_app or sim.handle_appstorage
  • tagName: string that identifies the data. An empty string will remove all custom data
  • data: your custom data. If nullptr or empty, the current data under the specified dataName will be removed
  • dataSize: size of your custom data

Return

  • -1 if operation was not successful
